# Challenge 065 - Group Anagrams

## Problem Statement

Write a C++ program to group a list of strings into groups of **anagrams**.

Two strings are anagrams if one can be formed by rearranging the letters of the other.

## Requirements

- Accept the number of strings and the strings themselves.
- Group the strings that are anagrams together.
- Print each group of anagrams on a separate line.

## Example

### Input
```
Enter number of strings: 6  
Enter strings: eat tea tan ate nat bat
```
### Output
```
Group 1: eat tea ate  
Group 2: tan nat  
Group 3: bat
```
